# **App Name**: ServerWatch

## Core Features:

- Server Status Monitoring: Continuously monitor the online/offline status of specified server applications (e.g., Portainer, Netdata, n8n, Nextcloud).
- Project Status Monitoring: Continuously monitor the online/offline status of personal web apps (e.g., shop.armanautoxperts.in, gallery.armanautoxperts.in).
- Visual Status Indicators: Display the status of each server and project with clear green/red color-coded indicators.
- Auto-Refresh: Automatically refresh status indicators every 30 seconds to ensure up-to-date information.
- Tab Launcher: Enable users to open monitored apps and projects in new tabs with a single click.
- Alerting Tool: Use an LLM as a tool to generate proactive alerts and recommendations based on abnormal behavior it notices, to aid in quickly identifying and solving issues.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to convey stability and reliability.
- Background color: Very light blue (#F0F4FF), nearly white, for a clean, uncluttered look.
- Accent color: Purple (#7B1FA2) for interactive elements, providing a touch of sophistication.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and short body text, and 'Inter' (sans-serif) for longer descriptions, if needed.
- Use simple, outlined icons for server applications and projects.
- Header with a logo and 'Server Dashboard' title, followed by categorized sections for 'Server Apps' and 'My Projects'.
- Subtle fade-in and scale-up animations upon loading, complemented by smooth transitions using Framer Motion.
