# ServerWatch

A Next.js dashboard to monitor the status of your server applications and personal projects.

## Features

- Real-time status monitoring (online/offline).
- Auto-refreshes every 30 seconds.
- Click to open any service in a new tab.
- Proactive AI analysis to suggest solutions for offline services.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can configure the services to monitor in `src/lib/config.ts`.
